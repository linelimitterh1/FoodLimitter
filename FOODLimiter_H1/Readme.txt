# FOODLimiter

Lineで賞味期限管理

## 概要

Line botで食品の賞味期限を管理しする。
また、食品ロスを抑える。

## 機能

- 食品をデータベースに登録
- 登録した食品の一覧を表示
-- 使い切ったものを削除
- 期限内に消費した、または消費できなかったに応じて食品ロスへの貢献を視覚化

## 必要要件

- スマートフォンで使用すること
- Lineのアプリをインストールしていること
- FOODLimiterを友達登録すること


## 使い方

1. tourokuQR.pngをLineの友達追加でスキャン 
2. トーク画面で入力ページに行く
3. 送信後、ブラウザを閉じる
4. 一覧画面で入力結果を確認
5. 一覧画面で削除・レシピ検索
6. スコア画面で自分の貢献度を確認
7. 使わなくなったら友達から削除
