# FOODLimiter

Lineで賞味期限管理

## 概要

Line botで食品の賞味期限を管理しする。
また、食品ロスを抑える。

## 機能

- 食品をデータベースに登録
- 登録した食品の一覧を表示
-- 使い切ったものを削除
-- 一部の食品（牛乳，ヨーグルト）の場合開封ボタンで開封後期限に更新
-- 食材名のまま外部サイト(Rakutenレシピ)で検索
- 期限内に消費した、または消費できなかったに応じて食品ロスへの貢献を視覚化

## 必要要件

- スマートフォンで使用すること
- Lineのアプリをインストールしていること
- FOODLimiterを友達登録すること


## 使い方

1. tourokuQR.pngをLineの友達追加でスキャン 
2. トーク画面で入力ページに行く
3. 送信後、ブラウザを閉じる
4. 一覧画面で入力結果を確認
5. 一覧画面で削除・レシピ検索
6. スコア画面で自分の貢献度を確認
7. 使わなくなったら友達から削除

## 各ファイルの説明
- public 静的データの格納場所，画像とスタイルシートが入っている．
- functions
-- node_modules：使用するモジュールが集まっている
-- routes：各ページのjavascriptの格納場所
-- views:各ページのテンプレートエンジンの格納場所

index:デバッグのページ遷移用
input:食材名と期限の入力
continue_image:input後画像を入力するかの確認画面
input_image:画像を入力する画面
complete:画像登録の完了を知らせる画面
stocklist:入力したデータの確認画面
history:直近5件の入力履歴の表示画面
score:食材ロスの減少の度合いの可視化をする画面
